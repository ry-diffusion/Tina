# Tina (Alpha - very very unstable)

The only sane unofficial Whatsapp desktop app.

# Why should you use?
Its fucking native. No electron/tauri/webview bs.

# What features are currently implemented?

- [x] Statuses
- [x] Avatars
- [x] Contacts
- [x] Write Messages
- [x] Send stickers and media via chat
- [x] View Media (Video, sticker, images) in Chat
- [x] Channels (newsletter)
- [X] Groups
- [x] i18n
- [x] Chat Preview 
- [x] Phone Pairing

# What isn't a implemented?
- [ ] Group Details
- [ ] Reply to messages (sending works!)
- [ ] Reactions
- [ ] Contact Details
- [ ] Start new chat from a number
- [ ] Responsive Layout
- [ ] Database encryption (by password or hardware passkey)
- [ ] Proper Chat renderer (WA Markup features work, but still too unstable)
- [ ] Proper Memory Optimization
- [ ] _and a lot more_
      
# what you are using for the app?
- Libadwaita for UI
- Whatsmeow for the messaging backend

# motivations
Whatsapp doesn't have a fucking decent app for Desktop. I was using whatsapp in Waydroid because i got so pissed off of WA Web Optimizations (AND IT STILL NOT GREAT YET)
